## Produit Principal

### Outil Self-Hosted pour Agences Marketing/Digitales

**Problème Identifié**
Trous dans les campagnes marketing :
- Relances manquées
- Suivi incomplet
- Opportunités perdues
- Erreurs humaines

**Solution Proposée**
- Centralisation multi-canaux
- Automatisation intelligente + personnalisation
- Vue claire et unifiée
- Maintien proximité humaine avec client
- Minimisation perte de leads

**Positionnement**
- Compatible avec outils existants des agences
- Peut partir de zéro si besoin
- Comble les manques humains dans le processus
