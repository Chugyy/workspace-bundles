## Introduction

Un frontend bien conçu est essentiel pour offrir une expérience utilisateur optimale, maintenable et performante. Ce guide compile les meilleures pratiques appliquées dans le projet Autom-Goodz AI pour vous aider à construire des interfaces modernes avec Next.js 16, React 19, Tailwind CSS v4 et TypeScript.

### Pourquoi les bonnes pratiques sont importantes

Un frontend bien structuré offre plusieurs avantages :
- **Maintenabilité** : Code clair et organisé, facile à faire évoluer
- **Performance** : Temps de chargement optimisés, expérience fluide
- **Accessibilité** : Interface utilisable par tous
- **Cohérence visuelle** : Design system unifié
- **Developer Experience** : Développement rapide et agréable

### Stack technique du projet

- **Framework** : Next.js 16 (App Router) + React 19
- **Styling** : Tailwind CSS v4
- **UI Components** : shadcn/ui (Radix UI + Tailwind)
- **State Management** : React Query 5 + Context API
- **Forms** : React Hook Form + Zod
- **Theme** : next-themes (dark/light mode)
- **Drag & Drop** : @dnd-kit
- **Typography** : next/font (Outfit + Caveat)

### Objectifs de ce guide

Ce guide vise à documenter les choix architecturaux et les patterns appliqués dans le projet, permettant ainsi de :
1. Maintenir la cohérence du code
2. Faciliter l'onboarding de nouveaux développeurs
3. Éviter les erreurs courantes
4. Garantir la qualité et la performance
